CVAT Annotation Project

Object detection dataset creation for fruit ripeness classification using CVAT.

This project focuses on building a high-quality labeled dataset using CVAT for training an object detection model. The goal is to classify fruit ripeness levels such as ripe, unripe, and overripe using bounding box annotations.
