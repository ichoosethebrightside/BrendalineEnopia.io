About the Project

This portfolio showcases my work in building a labeled dataset using CVAT for object detection. The dataset focuses on identifying fruit ripeness levels including ripe, unripe, and overripe categories.

The goal is to support machine learning models for agricultural automation and quality inspection systems.

Project Overview

  • Designed annotation schema for fruit detection
  • Used bounding box annotation for object detection
  • Prepared dataset for YOLO model training
  • Ensured labeling consistency and accuracy

Annotation Process

  • Created tasks in CVAT
  • Defined labels: ripe, unripe, overripe
  • Drew bounding boxes around fruit objects
  • Validated annotation quality
  • Exported dataset for ML pipeline
